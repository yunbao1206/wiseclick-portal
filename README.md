# WiseClick Portal

> Free online tools for smarter decisions.

**Live site:** [wiseclick.site](https://wiseclick.site)

## Tools

| Site | Description |
|:-----|:------------|
| [Finance Calculators](https://calc.wiseclick.site) | 12 financial calculators — mortgage, loan, compound interest, 401k, tax, paycheck (all 50 US states) |
| [Insurance Calculators](https://insure.wiseclick.site) | 8 insurance calculators — auto, life, home, health, pet, travel insurance estimator |
| [Health & Fitness Calculators](https://fit.wiseclick.site) | 10 health calculators — TDEE, BMI, body fat, macros, calorie deficit, protein, BMR |

## Tech

- Pure HTML/CSS/JavaScript — no frameworks
- Hosted on GitHub Pages
- All calculations run in-browser (privacy-first)
- JSON-LD structured data for SEO
